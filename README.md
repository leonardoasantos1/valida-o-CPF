# Validação de CPF

Este projeto é um campo de entrada de CPF com formatação automática e validação simples. O usuário pode inserir um número de CPF, que será automaticamente formatado e validado visualmente.

## Funcionalidades

- Formatação automática do número de CPF ao digitar.
- Validação visual do número de CPF.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
